# Authoring Results

Save unblinded writing subagent outputs here.

Recommended path:

`comparing/authoring_results/<run_id>/<condition_id>_<replicate_id>.md`

Example:

`comparing/authoring_results/run-2026-05-01-D001/C1_R1.md`

Every article file should begin with the metadata block required by
`prompts/writing_subagent_prompt.md`. Do not edit generated prose after saving
unless the edit is logged in `../evaluation_log.md`.
